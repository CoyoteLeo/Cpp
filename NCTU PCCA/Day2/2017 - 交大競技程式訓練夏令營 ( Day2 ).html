2017 - 交大競技程式訓練夏令營 ( Day2 )

A

題目敘述：
你有一棵樹。你想要拔掉任意數量的邊，使得剩下的連通分量的大小的乘積最大。問最大可以是多少。

題解：
考慮一條鍊的情況，資料型態至少要能處理 2^350 數字的乘法(大數)。
不失一般性地將 00 設為根，轉為有根樹上的問題。
dp[u][0]dp[u][0]: 以 uu 為根的子樹，在 uu (以及 uu 為端點的所有邊) 被刪除的前提下，按題意操作後的最大乘積。
dp[u][1]dp[u][1]: 以 uu 為根的子樹，按題意操作後的最大乘積。
dp[u][0]=∏v ∈ child(u)dp[v][1]dp[u][0]=∏v ∈ child(u)dp[v][1]

觀察小結論：
最佳解一定不存在任一條長度大於 22 的路徑。
反證 => 如果長度是 33 以上，對半之後相乘一定更大。

所以最佳解只存在三種連通分量：
1. 一點獨立
2. 一個點，以及若干個子節點
3. 一個點 uu，一個 uu 的子節點 vv，以及若干個 vv 的子節點
第一種情況是遞迴的邊界條件。
第二種情況，uu 的哪些子節點 vv 要和 uu 合併，可以依照 dp[v][0] / dp[v][1]dp[v][0] / dp[v][1] 單調由大到小添加 (這個比值越大，代表和 uu 合併後，對整體的損失越小)。
第三種情況和第二種同理。

給你一堆字串，玩接龍，要按照順序，問接完之後頭尾相等的最大字串長度。
如果沒按照順序該怎麼做呢O_O? (-> 網路流(?))

題解：
將a、b、c、d轉成0、1、2、3
dp[i][j]存可以組成i開頭j結尾字串的最大長度
最後再去找所有dp[i][i]值最大的。

C

題目敘述：
求最大子序列，要是奇數

題解:
先求最大子序列的值（就是把所有正數加起來），若為奇數，即為答案
若為偶數，就扣掉絕對值最小的奇數

D

題目敘述:
求矩形覆蓋面積 會有小數點

題解：
掃描線，X照大小排後，去計算每個矩形加入及離開後會有多長的區間被覆蓋到乘上X的差，因為n很小所以可以隨便維護一下當前有覆蓋的地方，不用寫到線段樹甚麼的，然後要用double，float精度不夠。

E

題目敘述 :
有一串數字，一次可以消掉一串回文，求整串消掉至少需要幾次

題解 :
區間 DP

dp[i][j]dp[i][j] 紀錄消除第i項到第j項需要的次數
若s[i]=s[j]s[i]=s[j]，消中間最後一項時 s[i]s[i] 和 s[j]s[j] 會一起被消掉，所以 dp[i][j]dp[i][j] 可為 dp[i+1][j−1]dp[i+1][j−1]
最後枚舉斷點k求最小值 dp[i][j]=min(dp[i][j],dp[i][k]+dp[k+1][j])dp[i][j]=min(dp[i][j],dp[i][k]+dp[k+1][j])

F

題目敘述：
給一棵樹，樹上每個節點有value 並以1為root
現在要選一些點，選點的規則是每一個被選取的點，他的所有子樹裡被選取的點的數量要為偶數
求選出的點最大value和

題解：
樹DP
每一個點維護以這個點為根拿到奇數個點的最大value跟偶數個點的最大value
奇數個點的最大value為 所有子樹合起來為偶數個點的最大value+上自己 或是所有的子樹加起來為奇數個點的最大value自己不取
偶數個點的最大value為 所有子數合起來為偶數個點的最大value

G

題目敘述:
各種區間操作
區間加值
增加一個點
移除一個點
區間反轉
區間rotate
區間min

題解:
treap 懶人標
加值:切出 l~r 後懶人標
增加點:把要加入的地方切開後 merge 起來 ex:merge(a,merge(new node§,b))
移出點:把要移除的點切出來之後前後合併
區間反轉：lazy tag 在split時候要注意先反轉在去計算要切的地方
區間rotate: 切出l~r後 rotate k次就是切出l~r-k+1 r-k+1~r兩個交換
區間min: 切出l~r後 答案就出現了

dp

會經過的條件是xj<=xiandyj<=yixj<=xiandyj<=yi
dp[i]=C(xi+yi,yi)−∑i−1j=0dp[j]∗C(xi−xj+yi−yj,xi−xj)dp[i]=C(xi+yi,yi)−∑j=0i−1dp[j]∗C(xi−xj+yi−yj,xi−xj)
